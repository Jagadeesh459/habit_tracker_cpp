# C++ Habit Tracker

## Overview
A command-line application built in C++ to help users create habits, track daily completion, and maintain streaks with persistent storage.

## Features
- Add new habits
- Mark habits as completed for the day
- View all habits with current streaks
- Save habit data to a file
- Load habit data on program start

## Tech Stack
- C++
- STL (vector)
- File I/O (text-based)
